# 근현대 미술 3D 가상 갤러리 콘텐츠 기획서

## 1. 프로젝트 개요
- 프로젝트명: `Echoes of Modernity: 근현대 미술, 방을 거닐다`
- 형식: PC Windows 기반 1인칭 3D 가상 전시
- 주요 등장 인물: 플레이어, 큐레이터
- 핵심 경험: 여러 개의 방으로 이어진 갤러리를 직접 걸으며 작품을 감상하고, `E` 키 상호작용으로 작품 해설과 확장 정보를 탐색하는 체험형 전시

## 2. 시나리오 기획

### 2-1. 전시 컨셉 및 기획 의도
이 전시는 근현대 미술의 변화 과정을 "시대의 방을 통과하는 여정"으로 재해석한다. 관람객은 하나의 연표를 읽는 대신, 서로 다른 미감과 사회적 맥락을 지닌 방들을 직접 이동하며 시대별 감수성의 차이를 체험한다. 전시의 핵심 메시지는 "예술은 시대를 기록하는 감각적 언어"이며, 감상자가 수동적인 관객이 아니라 공간 속 탐험자가 되도록 설계한다.

기획 의도는 다음과 같다.
- 미술사 지식을 모르는 사용자도 공간 분위기와 큐레이터 해설만으로 흐름을 직관적으로 이해하게 한다.
- 작품 감상, 시대 맥락, 인터랙션을 결합해 교육성과 몰입감을 동시에 확보한다.
- 물리적 전시에서 어려운 조명 연출, 확대 정보, 장면 전환을 디지털 공간에서 적극 활용한다.

### 2-2. 타겟 관람객 정의
- 1차 타겟: 20~40대 일반 관람객 및 문화생활 수요층
- 2차 타겟: 미술관 방문 전 사전 학습이 필요한 학생, 교사, 교육 프로그램 참여자
- 3차 타겟: 오프라인 전시에 접근이 어려운 원격 사용자, 디지털 전시 포맷에 익숙한 게이머형 사용자

타겟 특성은 다음과 같다.
- 깊은 전공 지식보다 직관적 이해와 몰입형 체험을 선호한다.
- 짧은 텍스트보다 공간 변화, 오디오 해설, 선택형 팝업을 통한 정보 습득에 익숙하다.
- 키보드 이동과 간단한 상호작용에는 익숙하지만 복잡한 조작은 선호하지 않는다.

### 2-3. 공간 동선 계획
전시는 총 5개 구역으로 구성한다.

1. 프롤로그 홀
- 역할: 전시 전체 메시지 소개, 조작 안내, 큐레이터 첫 인사
- 연출: 어두운 로비에 대형 타이포 월과 은은한 스포트라이트 배치

2. Room A. 태동의 방
- 주제: 근대 전환기와 새로운 시선의 등장
- 연출: 뉴트럴 톤 벽면, 작품 간 간격을 넉넉히 둔 입문형 구성

3. Room B. 도시와 감정의 방
- 주제: 산업화, 도시화, 개인의 내면
- 연출: 대비가 강한 조명, 사운드 레이어, 영상 작품 병행 배치

4. Room C. 실험과 확장의 방
- 주제: 추상, 혼합매체, 설치적 감각
- 연출: 방 중앙에 조형 오브제와 프로젝션 요소를 두어 체류 시간을 유도

5. 에필로그 라운지
- 역할: 전시 요약, 인상 깊은 작품 다시 보기, 종료 메시지
- 연출: 밝은 조도와 휴식형 UI, 작품 북마크 요약 패널 제공

관람 흐름은 다음과 같다.
- 진입: 플레이어는 프롤로그 홀에서 기본 조작과 전시 목적을 익힌다.
- 몰입: 각 방은 좁은 연결 통로로 이어져 장면 전환의 리듬을 만든다.
- 집중: 방마다 대표 작품 1점은 정면 접근 동선 끝에 배치해 시선 정착점을 만든다.
- 회고: 에필로그에서 방별 핵심 키워드를 다시 제시해 감상 경험을 정리한다.

### 2-4. 사용자 인터랙션
공간 인터랙션은 "가까이 가서 `E` 키를 누른다"는 단일 규칙으로 통일한다.

- 작품 감상: 작품 앞 인터랙션 존에 진입하면 `E 키로 작품 해설 보기` 프롬프트 표시
- 큐레이터 해설: `E` 키 입력 시 큐레이터 보이스 스크립트와 자막 팝업 재생
- 심화 정보: 한 번 더 `E` 키 입력 시 작가, 제작연도, 매체, 시대 배경 등 상세 메타데이터 노출
- 공간 장치 작동: 특정 지점에서 `E` 키로 조명 변화, 영상 재생, 다음 방 안내 연출 실행
- 종료 UX: 팝업 상태에서 `E` 또는 닫기 버튼으로 해설 종료 후 다시 자유 이동

인터랙션 설계 원칙은 다음과 같다.
- 정보는 한 번에 과도하게 주지 않고 1차 해설, 2차 심화 정보로 나눈다.
- 플레이 흐름을 끊지 않도록 팝업은 화면 중앙이 아닌 측면 패널 중심으로 구성한다.
- 모든 주요 인터랙션에는 큐레이터의 짧은 유도 문장을 붙여 행동 목적을 명확히 한다.

### 3-2. 공간 테마 및 배치
| 구역 | 색상 테마 | 조명 방향 | 배치 전략 |
| --- | --- | --- | --- |
| 프롤로그 홀 | 차콜, 웜 화이트, 브라스 | 천장 스포트 + 바닥 라인 조명 | 중앙 타이포 오브제를 기준으로 원형 진입 |
| Room A | 아이보리, 세이지, 우드 | 균일한 확산광 | 벽면 감상형 배치로 안정감 확보 |
| Room B | 네이비, 버건디, 스틸 그레이 | 강한 명암 대비, 측면광 | 도시 이미지와 영상 작품을 교차 배치 |

| 에필로그 | 소프트 그레이, 샌드, 골드 | 밝은 확산광 | 체류형 라운지와 요약 UI 결합 |

배치 원칙:
- 각 방의 첫 작품은 해당 시대 감수성을 대표하는 앵커 작품으로 설정한다.
- 텍스트 월은 입구 좌측, 작품은 정면 또는 우측으로 두어 시선 흐름을 일정하게 유지한다.
- 영상/오디오 작품은 정적 회화 작품 사이에 배치해 리듬을 만든다.

### 3-3. 스토리텔링 및 텍스트

#### 도슨트 스크립트 톤앤매너
- 한 문장 길이는 짧고 명확하게 유지
- 미술사 용어는 풀어 설명
- 감정적 단서를 먼저 주고, 뒤에 정보 제공

#### 예시 도슨트 스크립트
- 프롤로그: "지금부터 우리는 작품을 보는 것이 아니라, 시대의 감각이 어떻게 바뀌는지를 직접 걸으며 만나게 됩니다."
- Room A 대표작: "이 작품은 익숙한 실내 풍경을 그렸지만, 시선의 방향은 이전 시대보다 훨씬 개인적입니다."
- Room B 대표작: "도시는 더 화려해졌지만, 화면 속 인물의 감정은 오히려 더 고립되어 보입니다."
- Room C 대표작: "형태가 부서진 자리에서 작가는 감정보다 감각의 흔적을 먼저 남깁니다."
- 에필로그: "방을 지나며 본 변화는 예술의 변화이면서 동시에 우리를 바라보는 시선의 변화이기도 합니다."

#### 월 텍스트 예시
- 프롤로그 월: "근현대 미술은 단순히 양식이 바뀐 역사가 아니라, 세계를 보는 방식이 달라진 기록이다."
- Room A 월: "새로운 시대는 먼저 낯선 시선으로 시작된다."
- Room B 월: "도시의 속도는 인간의 내면을 더 선명하게 드러낸다."
- Room C 월: "형태가 사라질수록 감각은 더 또렷해진다."

#### 팝업 안내문 예시
- 기본 프롬프트: "`E` 키를 눌러 작품 해설을 확인하세요."
- 심화 정보 프롬프트: "`E` 키를 다시 눌러 작가와 시대 정보를 확인하세요."
- 이동 안내: "다음 방으로 이동하면 시대의 분위기가 크게 전환됩니다."

### 3-4. 기술 사양
- 1차 지원 플랫폼: Windows PC
- 확장 가능 플랫폼: WebGL 기반 웹 전시, 키오스크 모드
- 권장 해상도: 1920x1080 이상
- 입력 장치: 키보드 + 마우스
- 핵심 인터랙션 키: 이동 `WASD`, 시점 `Mouse`, 상호작용 `E`
- 오디오: 스테레오 헤드폰 및 기본 스피커 환경 대응
- 이미지 에셋 권장: 2K 텍스처, 대표 작품은 4K 원본 보관
- 영상 에셋 권장: H.264 MP4, 1080p, 루프 재생 최적화

## 4. 제작 메모
- 큐레이터는 실재 인물보다 "공간을 해석해 주는 목소리"에 가깝게 설정하면 몰입이 높다.
- 각 방 입구에 1문장 키 메시지를 고정 배치하면 사용자가 길을 잃어도 서사를 놓치지 않는다.
- 향후 구현 단계에서는 작품별 트리거 범위, 팝업 UI 프리팹, 오디오 믹싱 규칙을 별도 설계 문서로 분리하는 것이 좋다.
