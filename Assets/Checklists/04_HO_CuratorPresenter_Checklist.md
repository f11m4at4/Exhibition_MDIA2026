- [x] 클래스와 주요 함수에 역할 설명 주석이 있다.
- [x] 현재 내레이션 대상과 단계 상태를 관리한다.
- [x] 첫 호출, 다음 호출, 종료 호출 흐름이 분리되어 있다.
- [x] 빈 내레이션 데이터에서 UI를 열지 않고 로그를 남긴다.
- [x] `CuratorVisualDummy`, `Capsule`, `NavMeshAgent`의 활성화나 설정값을 런타임에 자동으로 바꾸지 않는다.
- [x] `NavMeshAgent`는 인스펙터와 씬에 미리 설정된 값을 그대로 사용한다.
- [x] 플레이어 앞 2미터 목표가 NavMesh 밖이면 플레이어 위치를 fallback 목적지로 다시 샘플링한다.
- [x] `Ensure` 접두사 함수 없이 현재 동작이 구성되어 있다.

- [x] 큐레이터 이동 상태를 `Idle`/`Move`로 구분하고, `Idle` 2초 경과 후 `Move` 전환 시 `Move` 트리거가 활성화되도록 반영했다.
- [x] 목표 지점 도착 시 상태를 `Explain`으로 전환하고, 이동을 멈춘 뒤 `Explain` 트리거가 1회 활성화되도록 반영했다.
