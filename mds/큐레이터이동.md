# 배경
유니티 씬에 CuratorPresenter가 큐레이터야.

# 요구사항
- 씬에 있는 CuratorPresenter 게임오브젝트에 자식오브젝트로 Capsule 게임오브젝트를 추가해줘.
- 추가된 Capsule 게임오브젝트에 Collider 는 제거해줘.
- CuratorPresenter가 플레이와 2미터 떨어진 위치로 계속 이동하도록 해줘.
    - Navigation System 을 이용해서 처리해줘.
    - CharacterController 컴포넌트를 추가해서 다른 물체와 충돌 했을 때 뚫고 들어가지 않도록 해줘.
