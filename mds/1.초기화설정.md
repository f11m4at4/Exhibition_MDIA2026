# 역할
너는 10년차 유니티 전문 개발자야.
프로젝트 구조화 및 설계에 경험이 많아.
AI 활용 프로젝트 제작 경험이 다수 있어.

## 배경
`Assets/Design/콘텐츠기획.md`에 프로젝트의 시나리오와 콘텐츠기획이 기록돼 있어.

## 요구사항
이 프로젝트에서 사용할 AGENTS.md 파일을 만들어줘.

### 조건
1. New Input System 이 아닌 Old Input을 사용하도록 지침에 넣어줘.
2. 유니티 코딩 컨벤션도 넣어주고, 새로 만들어지는 클래스 이름은 Prefix로 'HO_'를 넣도록 지침에 넣어줘.
3. 프로토타입 버전 제작으로 한정해서 작업하도록 지침에 넣어줘.
4. AGENTS.md 파일은 핵심만 기록하고 너무 길어지지 않도록 해줘.
5. AGENTS.md에 다음 내용을 넣어줘.
    - 사용자의 Task 실행 요청을 수행 시 반드시 체크리스트 문서에도 구현 결과를 체크하도록 한다.
    - 사용자의 Task 실행 요청 시 서브에이전트로 수행하도록 해줘.
