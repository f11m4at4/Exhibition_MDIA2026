# 역할
너는 10년차 전시기획 콘텐츠 기획자야.

## 배경
3D 공간에 여러개의 방으로 구성된 갤러리에 근현대 미술 작품을 전시하는 3D 가상공간 프로젝트를 진행하려고해.
- 실행 환경: PC Windows
- 등장 인물: 플레이어, 큐레이터

## 요구사항
이 프로젝트를 만들기위해 시나리오 기획, 콘텐츠 기획서를 작성하려고해.
`Assets/Design` 폴더의 `콘텐츠기획.md`에 다음 내용을 작성해줘.

1. 시나리오 기획:
- 전시 컨셉 및 기획 의도: 전시의 핵심 메시지와 목적 정의
- 타겟 관람객 정의: 누구를 위한 전시인지 분석
- 공간 동선 계획: 진입 동선, 관람 흐름, 퇴장까지의 여정 설계
- 사용자 인터렉션: 관람객이 공간 내에서 유도할 행동(키보드의 `E`키 Press)

2. 콘텐츠 기획:
- 작품 라인업(스펙): 전시할 3D 모델, 이미지, 영상, 오디오 등의 목록 및 메타데이터
- 공간 테마 및 배치: 섹션별 공간 디자인 모드(색상, 조명)와 작품 배치 위치
- 스토리텔링 및 텍스트: 작품 설명(도슨트 스크립트), 월 텍스트, 팝업 안내문
- 기술사양: 지원 플랫폼(Windows, 웹 등)
- 콘셉트 이미지 / 무드보드
    - 역할: 공간의 전체적인 분위기, 색감, 조명 느낌을 하눈에 보여주는 이미지 모음.
    - 포함 내용: 레퍼런스 인테리어 사진, 사용할 색상 팔래트, 텍스트 폰트 스타일, 조명 효과 예시 등.
    - 실제 콘셉트 이미지 생성해서 `Assets/Concept`에 파일저장
- 공간 레이아웃 / 스토리보드
    - 역할: 관람객의 시선 이동에 따른 화면 연출과 인터랙션의 흐름을 시각적으로 설계하는 지도.
    - 포함 내용: 전시 공간의 평면도(2D Layout), 주요 화면의 스케치나 와이어프레임, 클릭시 팝업이 뜨는 연출 과정
    - 실제 스토리보드 이미지 한장에 전체 장면이 들어 있는 파일 하나와 각 장면별로 생성해서 `Assets/Storyboard`에 순서 번호와 장면 제목을 파일 이름으로 만들어 파일저장
